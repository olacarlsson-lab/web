KONST / iOS-APP

KONST­APPEN

Utforska Västra Götalandsregionens konstsamling – tusentals verk i din ficka.

HÄMTA PÅ APP STORE →

FUNKTIONER

01

SÖK

Sök på konstnär, titel, plats eller teknik i hela VGR:s samling om tusentals verk.

02

UTFORSKA

Bläddra bland konstverk, se bilder och läs detaljerad information om varje objekt.

03

FILTRERA

Filtrera samlingen efter teknik, år, konstnär eller geografisk plats.

04

GRATIS

Helt gratis att hämta och använda. Ingen registrering, inga prenumerationer.

OM APPEN

Konstappen är ett inofficiellt verktyg för att utforska Västra Götalandsregionens offentliga konstsamling. Data hämtas från VGR:s öppna API och presenteras i ett enkelt, snabbt och tillgängligt gränssnitt.

Appen är skapad av Ola Carlsson som ett fritidsprojekt i syfte att göra kulturarvet mer tillgängligt för allmänheten.
