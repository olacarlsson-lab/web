ANVÄNDAR­VILLKOR

SENAST UPPDATERAD / MARS 2026

GODKÄNNANDE AV VILLKOR

Genom att ladda ner eller använda Konstappen godkänner du dessa användarvillkor. Om du inte godkänner villkoren ska du inte använda appen.

OM APPEN

Konstappen är ett inofficiellt, icke-kommersiellt projekt utvecklat av Ola Carlsson. Appen är inte knuten till eller sponsrad av Västra Götalandsregionen (VGR). Data om konstverk hämtas från VGR:s öppna API och presenteras i informationssyfte.

TILLÅTEN ANVÄNDNING

Du får använda appen för personliga och icke-kommersiella ändamål. Det är inte tillåtet att:

  • Använda appen på ett sätt som bryter mot tillämplig lag
  • Försöka extrahera, kopiera eller vidaredistribuera appens källkod
  • Använda appen för att masshämta eller scrapa data från VGR:s API i strid med VGR:s villkor
  • Anpassa, modifiera eller skapa derivatverk baserade på appen utan tillstånd

INNEHÅLL OCH UPPHOVSRÄTT

Information och bilder om konstverk tillhör respektive rättighetsinnehavare. VGR tillhandahåller data via sitt öppna API under egna villkor. Konstappen gör inga anspråk på äganderätten till detta innehåll.

Appens gränssnitt, design och kod ägs av Ola Carlsson.

TILLGÄNGLIGHET OCH ÄNDRINGAR

Appen tillhandahålls i befintligt skick ("as is"). Vi garanterar inte att appen alltid är tillgänglig, felfri eller fullständig. Vi förbehåller oss rätten att ändra, pausa eller avsluta appen när som helst utan förvarning.

VGR:s API är en extern tjänst och vi ansvarar inte för dess tillgänglighet eller riktigheten i dess data.

ANSVARSBEGRÄNSNING

Ola Carlsson ansvarar inte för direkta eller indirekta skador som uppstår vid användning av appen eller vid avbrott i tjänsten. Appen används på eget ansvar.

TILLÄMPLIG LAG

Dessa villkor regleras av svensk lag. Eventuella tvister ska avgöras i svensk domstol.
